how do i vote?

simple! next to each character there is a dropdown where you can select what placement, out of 20, you would give each character. basically, for any given character you should give them a placement out of 20. 1st is the best, 20th is the worst.


vote for all characters. sorry, i don't make the rules (just kidding i do). also, i suggest checking the checkboxes after you've ranked the characters. kinda gets messy otherwise.


there are to be no ties, no two characters may have the same rating.


once you're done ranking the characters, you then can give them a tick or not depending if you would be happy if they were crowned best character in project sekai. if you don't, then don't tick their name. this is transitive, as in if you tick a character's name it'll automatically tick any character ranked above and de-tick any character ranked below.
